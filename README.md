# Serverhop Libary
Modular Roblox server hopping library with dynamic filtering support.

## Installation

Require the module in your script:

```lua
local SERVER_HOP = loadstring(game:HttpGet("https://raw.githubusercontent.com/s-eths/serverhop-library/main/src.lua"))();
```
